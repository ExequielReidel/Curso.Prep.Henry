* Variables: Una variable es un palabra en la que se pueden guarda cosas, para despues llamar a esas cosas por esa palabra.

* Strings: Son todo lo relacionado con el texto, o letras unidas como una cadena formando palabras 

* Funciones (argumentos, `return`): Son objetos que contienen formulas las cuales podemos inbocar para hacer algo con las variables, como sumarlas, restarlas, etc.

* Declaraciones `if`: Es como una pegunta, que sirve para decir "¿si es verdad?", y si determina verdadera la condicion que le demos va a ejecutar el codigo que le asignemos.

* Valores booleanos (`true`, `false`): verdadero o falso, cierto o no, si o no, sirven para determinar si algo es verdad o no.
